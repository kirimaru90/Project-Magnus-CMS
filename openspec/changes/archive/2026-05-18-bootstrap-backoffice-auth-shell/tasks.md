## 3. Theme system

    - `theme = signal<'light' | 'dark'>(<initial>)` where the initial value is computed at construction time as: `localStorage.getItem('rc.bo.theme')` if set to `'light'` or `'dark'`, else `matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light'`.
